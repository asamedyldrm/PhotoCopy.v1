# PhotoCopy.v1

Kopyalayıp sadece isimlerini değiştirmem gereken bir sürü fotoğraf dosyası için yazmış olduğum program. <br>
Bu sayede kullanıcıdan istenilen fotoğraf alınır ve istediği isimler kadar kopyalanır, isimlendirilir.
